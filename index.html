Most Popular Commands

Essential commands every DevOps engineer should know

ls -lah

List all files with detailed information in human-readable format

Usage: ls -lah /path/to/directory

grep -r

Search for text patterns recursively in directories

Usage: grep -r "pattern" /path/to/search

systemctl

Control systemd services and check system status

Usage: systemctl status service-name

docker ps

List running Docker containers with status information

Usage: docker ps -a (show all containers)

kubectl get

List Kubernetes resources like pods, services, and deployments

Usage: kubectl get pods -n namespace

git status

Check the current state of your Git repository

Usage: git status -s (short format)
